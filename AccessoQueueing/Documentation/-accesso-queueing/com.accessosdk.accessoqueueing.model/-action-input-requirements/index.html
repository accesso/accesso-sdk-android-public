ActionInputRequirements

data class ActionInputRequirements(val id: String, val dataType: String, val isRequired: Boolean, val description: String, val title: String, val defaultValue: Int, val minimum: Int, val maximum: Int)

The ActionInputRequirements data class holds the specification and constraints for the input required by a ReserveAction. It describes the nature of the data expected, its bounds, and whether or not it is a required parameter for the action.

See also

ReserveAction

for the details of the reserve action associated with the input requirements.
