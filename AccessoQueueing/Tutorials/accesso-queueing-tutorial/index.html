Installation

  1. Place the AccessoEntitlements-release.aar file within the app/libs directory.
  2. Include a dependency to the the SDK in the module-level build.gradle file and add required dependencies
    implementation(files("libs/AccessoQueueing-release.aar"))
    implementation("com.squareup.retrofit2:retrofit:2.9.0")
    implementation("com.squareup.retrofit2:converter-gson:2.9.0")
  1. Add required permissions and replace theme.
    <manifest>
        <uses-permission android:name="android.permission.INTERNET" />
        <application>
            tools:replace="android:theme"
        </application>
    </manifest>
  1. Sync gradle.
  2. Add accesso-services.json to the assets directory with all fields filled out
{
   "VENUE_ID": "",
   "APP_ID": "",
   "FIREBASE_CONFIG": "google-service.json",
   "BASE_URL": "https://host.te2.biz",
   "PREFERRED_LANGUAGE": "en",
   "IDENTITY_USERNAME": "",
   "IDENTITY_PASSWORD": "",
   "USERNAME": "",
   "PASSWORD": "", 
   "ENTITLEMENTS_BASE_URL": "https://host.te2.biz"
}
  1. Configure the AccessoCore SDK. This must be done before using any features of the SDK.
        AccessoCore
            .setApplicationContext(applicationContext)
            .setActivity(activity)
            .build()
  1. Create new instance of the AccessoQueueing class
    val accessoQueueing: AccessoQueueing = AccessoQueueing()

Get Attractions

    coroutineScope.launch {
        try {
            val attractions: List<Attraction> = accessoQueueing.getAttractions()
        } catch (e: Exception) {
            //handle exception
        }
}

Get Attraction by ID

    coroutineScope.launch {
        try {
            val attraction: Attraction = accessoQueueing.getAttractionById("ID123")
        } catch (e: Exception) {
            //handle exception
        }
}
