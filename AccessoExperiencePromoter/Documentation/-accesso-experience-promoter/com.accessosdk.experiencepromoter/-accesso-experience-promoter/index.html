startUpdatingLocation
Link copied to clipboard
open override fun startUpdatingLocation(distanceInMeters: Float, timeInterval: Long, activity: Activity)

startUpdatingLocation - Starts updating the devices location and sends location info to Accesso servers via a call to trackDeviceLocation. The location is updated and sent whenever the device moves at least distanceInMeters and on a regular interval set by timeInterval. If location permission was not given or was denied, this function will prompt for it again.
