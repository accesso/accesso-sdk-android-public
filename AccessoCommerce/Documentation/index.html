Developer User Guide for the Accesso Commerce SDK for Android

Installing the SDK

  1. Place the AccessoCommerce-release.aar file within the app/libs directory.

  2. Include a dependency to the the SDK in the module-level build.gradle file and add required dependencies

    implementation(files("libs/AccessoCommerce-release.aar"))
    implementation("com.squareup.retrofit2:retrofit:2.9.0")
    implementation("com.squareup.retrofit2:converter-gson:2.9.0")

  1. Add required permissions and replace theme.

    <manifest>
        <uses-permission android:name="android.permission.INTERNET" />
        <application>
            tools:replace="android:theme"
        </application>
    </manifest>

Setup SDK Configurations

  1. Place the accesso-services.json file within the app/src/main/assets directory

  2. The SDK will read the config files on initialization

Initializing the SDK

  • Before requesting commerce capabilities, initialize the SDK core with the required builder functions and create an instance of the AccessoCommerce class.

    AccessoCore
        .setActivity(activity)
        .setApplicationContext(context)
        .build()

        val accessoQueueing = AccessoCommerce()
Content copied to clipboard

  • ``applicationContext`` is the context of the host application

  • ``activity`` is the activity that initializes the SDK

  • ``build()`` must be called after setting configurations

Retrieving Commerce Store Products

  • After the Accesso Commerce SDK has been initialized, you can start calling methods needed for your application, like the showStoreProducts or getStoreProducts method.

  • To simply allow the SDK to make the API calls and handle the UI for you, just call showStoreProducts

    accessoCommerce.showStoreProducts(this)
Content copied to clipboard

  • Alternatively if you want more control, you can also call getStoreProducts to get the list directly.

  • Note ``getStoreProducts()`` is a suspend function so it must be called within a kotlin coroutine

coroutineScope.launch {
    try {
        val storeProducts: List<StoreProduct> = accessoCommerce.getStoreProducts()
    } catch (e: Exception) {
        //handle exception
    }
}

  • ``getStoreProducts()`` returns a ``List<StoreProduct>``
